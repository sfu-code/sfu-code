Rome wasn't built in a day.

But when motivated people come together, impressive things happen. That's why we're starting the SFU Software Club.

What is this?

SFU Code is a new club starting in Fall 2018. We aim to bring together all kinds of developers so knowledge can be shared, connections can be made, and projects can be built.
